Help & Frequently Asked Questions

üöÄGetting Started

Policible is super simple to use. Here's how to get going:

  1. Add a Profile: Go to the Profiles tab and tap "Add Profile" to create a profile for yourself or family members
  2. Add Your Policies: Head to the Policies tab and tap the + button to add your insurance policies
  3. Add Agents (Optional): Go to the Agents tab to save your insurance agent contact info
  4. Set Reminders (Optional): Open any policy and tap the bell icon to enable payment reminders

‚ùìFrequently Asked Questions

Is my data safe?

Yes! All your data is stored locally on your device. Nothing gets sent to any servers. We literally can't see your data even if we wanted to.

Do I need to create an account?

Nope! No sign-up, no email, no personal info needed. Just download and start using it.

How do I share policies with family?

Go to the Profiles tab, tap the share icon on any profile, select which policies to share, create a passcode, and share the encrypted file. The recipient will need to import it in their app using the same passcode.

How do payment reminders work?

Open any policy, tap the bell icon, and enable reminders. You'll get notified 2 weeks before each payment is due. When you pay, just mark it as "Paid Already" to schedule the next reminder.

Can I backup my data?

Yes! Go to Settings > Backup Data, create a passcode, and save the encrypted backup file. You can restore it anytime from Settings > Restore Backup.

What if I forget my backup passcode?

Unfortunately, we can't recover lost passcodes since everything is encrypted locally. Make sure to keep your passcode somewhere safe!

How do I enable biometric lock?

Go to Settings and toggle on "Biometric Authentication". You'll need to have Face ID or fingerprint set up on your device first.

What happens if I delete the app?

All your data will be deleted. Make sure to create a backup first if you want to keep your data!

Can I use this on multiple devices?

Each device stores its own data. To sync between devices, create a backup on one device and restore it on another.

Can I delete my data?

Absolutely! You have complete control over your data. Delete individual policies, profiles, or agents anytime. Want to wipe everything? Go to Settings > Delete Data. Since all data is stored locally on your device, you're in full control.

Is this app really free?

Yes! Free to use, no ads, no in-app purchases.

üîßTroubleshooting

Notifications not working?

  • Make sure you've granted notification permissions in your device settings
  • Check that the policy has a payment frequency other than "One-time"
  • Verify the start date is correct

Can't import a shared file?

  • Make sure you're using the correct passcode
  • Make sure you're importing from a sharing profile file (not a Backup Data)
  • Check that it's a .policible file

Biometric authentication not available?

  • Make sure Face ID or fingerprint is set up in your device settings
  • Make sure permission is granted for the app
